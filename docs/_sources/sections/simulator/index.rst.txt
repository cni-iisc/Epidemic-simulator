The Simulator 
============================
The simulator takes many input options and it is useful to know what these are so that you can decide which of those should be set to default values and which of those are to be specified.

In order to see the options, and a brief description of the options, you can just run the following command:

.. code-block:: bash

   (edaDev) $ ~/tutorial/markov_simuls/cpp-simulator > ./drive_simulator -h
